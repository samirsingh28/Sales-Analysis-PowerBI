## 📌 Project Description

This project is an end-to-end **Power BI capstone dashboard** built using the Sample Superstore dataset.

It transforms raw retail transactional data into a **clean, interactive, and insightful dashboard** that helps businesses:

* Monitor key performance indicators (KPIs)
* Analyze regional and category-wise performance
* Identify profit and loss areas
* Understand sales trends over time

---

## 🚀 Features

* 📊 KPI Cards: Total Sales, Total Profit, Total Orders, Profit Ratio
* 📈 Line Chart: Sales trend (monthly & yearly)
* 🌍 Bar Chart: Sales by Region
* 🛍️ Pie Chart: Category-wise distribution
* 📉 Profit Analysis: Sub-category performance (with conditional formatting)
* 🏆 Top 10 Products by Sales
* 🎛️ Interactive Slicers: Region, Category, Date
* 🧠 Advanced DAX Measures:

  * `DIVIDE()`
  * `DISTINCTCOUNT()`
  * `SAMEPERIODLASTYEAR()`

## 📊 Key Insights

* West Region contributes the highest revenue (~32%)
* Technology category yields the highest profit margin
* Tables & Bookcases result in consistent losses
* Peak sales occur during November–December
* 20–25% steady year-over-year sales growth
